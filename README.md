Un program care modelează un joc RPG care conține trei tipuri de luptători, respectiv magi, eroi și goblini, care se bat în trei tipuri de arene. Scopul codului este să calculeze rezultatul unei lupte folosind reguli de avantaj. Magul este mai bun decât eroul, eroul este mai bun decât goblinul, iar goblinul este mai bun decât magul. Luptătorul care are avantaj de clasă primește un bonus de zece la sută la puterea finală. De asemenea, fiecare arenă oferă un bonus luptătorului din aceeași categorie și îi penalizează pe ceilalți. Puterea se calculează adunând viața cu damage-ul înmulțit cu zece, la care se adaugă sau se scad bonusurile de teren și de clasă.
